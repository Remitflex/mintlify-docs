This guide walks through a complete swap flow: get an API key, check the rate, create a swap, and poll until it completes. Examples use the test key prefix `rmf_test_`.

## Prerequisites

- A RemitFlex account at [dashboard.remitflex.io](https://dashboard.remitflex.io)
- `curl` or Node.js
- For local testing: API running at `http://localhost:4000`

<Note>
  API keys are created in the dashboard (JWT login required). You cannot create keys via the public API.
</Note>

## Step 1 ÔÇö Create an API key

1. Log in to the dashboard (email + OTP).
2. Go to **Settings Ôćĺ API Keys** Ôćĺ **Create new key**.
3. Environment: **test**
4. Scopes: at minimum `fx:read`, `transfers:read`, `transfers:write`
5. Copy the key and export it:

```bash
export REMITFLEX_API_KEY="rmf_test_YOUR_KEY_HERE"
export API_BASE="https://api.remitflex.io/v1"
# Local: export API_BASE="http://localhost:4000/v1"
```

## Step 2 ÔÇö Fetch the rate

```bash
curl -s "$API_BASE/rates/USDC/EURC" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq .
```

Example response:

```json
{
  "status": "success",
  "message": "Rate fetched",
  "data": {
    "from": "USDC",
    "to": "EURC",
    "rate": 0.92,
    "fetchedAt": "2026-06-26T12:00:00.000Z"
  }
}
```

## Step 3 ÔÇö Create a swap

Mutating API-key requests require an idempotency key:

```bash
curl -s -X POST "$API_BASE/swaps" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" \
  -H "Idempotency-Key: $(uuidgen)" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "USDC",
    "to": "EURC",
    "amount": 10,
    "destinationAddress": "YOUR_SOLANA_WALLET_ADDRESS"
  }' | jq .
```

The response includes a `depositAddress` (ATA). Send the exact `from` token amount to that address before `expiresAt`.

```json
{
  "status": "success",
  "message": "Swap created. Deposit to the address provided.",
  "data": {
    "id": "cee5e1aa-e9f4-47cc-8f5c-870e1b90e3e3",
    "status": "awaiting_deposit",
    "depositAddress": "...",
    "expiresAt": "2026-06-26T12:30:00.000Z"
  }
}
```

## Step 4 ÔÇö Poll swap status

```bash
SWAP_ID="cee5e1aa-e9f4-47cc-8f5c-870e1b90e3e3"

curl -s "$API_BASE/swaps/$SWAP_ID" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq '.data.status'
```

Status progression:

```
awaiting_deposit Ôćĺ deposit_received Ôćĺ swapping Ôćĺ completed
```

Terminal failure states: `failed`, `expired`, `cancelled`.

## Step 5 ÔÇö List transactions

```bash
curl -s "$API_BASE/transactions" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq '.data.transactions[:3]'
```
