RemitFlex is a stablecoin payments API for businesses that need programmatic access to swaps and cross-chain collection. Deposit stablecoins on Solana or across Relay corridors; RemitFlex handles routing, settlement, and ledgering.

## What you can do today

| Capability | API | Provider |
|------------|-----|----------|
| **Solana swaps** | `POST /v1/swaps` | Jupiter |
| **Cross-chain collection** | `POST /v1/payment-routes` | Relay |
| **FX rates** | `GET /v1/rates` | Jupiter (cached) |
| **Transaction ledger** | `GET /v1/transactions` | RemitFlex |
| **Customers** | `GET/POST /v1/customers` | RemitFlex |

<Note>
  Bank off-ramp and outbound merchant webhooks are planned but **not yet available** in the API. See [Off-ramp](/guides/off-ramp) and [Webhooks](/guides/webhooks) for the roadmap.
</Note>

## Getting started

<Steps>
  <Step title="Create your account">
    Sign up at [dashboard.remitflex.com](https://dashboard.remitflex.com) and complete OTP login.
  </Step>
  <Step title="Create an API key">
    In **Settings → API Keys**, create a test key with the scopes you need. Copy it immediately — it is only shown once.
  </Step>
  <Step title="Make your first call">
    Follow the [Quickstart](/quickstart) to fetch a rate and create a swap, or the [Send payment](/guides/send-payment) guide for payment routes.
  </Step>
  <Step title="Go live">
    Create a `rmf_live_` key, update your production secrets, and start routing real volume.
  </Step>
</Steps>
