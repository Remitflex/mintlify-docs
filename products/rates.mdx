Before creating an offramp, get a sell quote for the token amount and target currency. Remitflex caches prices and refreshes them in the background — reads are served from cache for low latency.

**Required scope:** `fx:read`

## Fiat sell rates (stablecoin → local currency)

```bash
curl "https://api.remitflex.io/v1/fiat/rates/USDC/100/NGN?network=base" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY"
```

Pass the quoted rate when calling `POST /v1/offramps`. See [Local fiat](/products/fiat-rails).
