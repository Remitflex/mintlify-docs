RemitFlex exposes two rate surfaces:

| Endpoint family | Use |
|-----------------|-----|
| `GET /v1/fiat/rates/...` | Stablecoin → local currency offramp quotes (PayCrest) |
| `GET /v1/rates/...` | Live Relay quotes for Solana EURC↔USDC before creating a [swap](/products/swaps) |

Fiat rates and swap rates are separate systems.

## Fiat sell rates (stablecoin → local currency)

```bash
curl "https://api.remitflex.io/v1/fiat/rates/USDC/100/NGN?network=base" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY"
```

## Solana swap rates (EURC ↔ USDC)

```bash
curl "https://api.remitflex.io/v1/rates/USDC/EURC?amount=100&tradeType=exact_input" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY"
```

Quotes are **live from Relay** and indicative only — `POST /v1/swaps` always re-quotes.

List pairs with `GET /v1/rates` and supported currencies with `GET /v1/rates/currencies`.
