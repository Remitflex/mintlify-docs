Solana token swaps are powered by Jupiter. You deposit one stablecoin; RemitFlex swaps and delivers the output token to your destination wallet.

## Prerequisites

- API key with `fx:read`, `transfers:read`, and `transfers:write`
- A Solana wallet address to receive swapped tokens

```bash
export API_BASE="https://api.remitflex.com/v1"
export REMITFLEX_API_KEY="rmf_test_YOUR_KEY"
```

## Step 1 — Check supported pairs

```bash
curl -s "$API_BASE/rates/currencies" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq '.data.pairs'
```

## Step 2 — Get the current rate

```bash
curl -s "$API_BASE/rates/USDC/EURC" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq .
```

Example response:

```json
{
  "status": "success",
  "message": "Rate fetched",
  "data": {
    "from": "USDC",
    "to": "EURC",
    "rate": 0.92,
    "fetchedAt": "2026-06-26T12:00:00.000Z"
  }
}
```

Rates are cached and refreshed in the background. If a pair is unavailable, you'll receive `503`.

## Step 3 — Create the swap

```bash
curl -s -X POST "$API_BASE/swaps" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" \
  -H "Idempotency-Key: $(uuidgen)" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "USDC",
    "to": "EURC",
    "amount": 100,
    "destinationAddress": "2dudFU32c5wsRpfRZDXBAJFirHC4hindqpKSCwwtDaAB"
  }' | jq .
```

| Field | Description |
|-------|-------------|
| `from` / `to` | Currency codes (e.g. `USDC`, `EURC`) |
| `amount` | Human-readable input amount |
| `destinationAddress` | Solana address to receive output tokens |

The response includes `depositAddress` (SPL token account) and `expiresAt`. Send the exact `from` token to `depositAddress` before expiry.

## Step 4 — Track status

```bash
SWAP_ID="your-swap-uuid"

curl -s "$API_BASE/swaps/$SWAP_ID" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" | jq '.data.status'
```

Status flow:

```
awaiting_deposit → deposit_received → swapping → completed
```

Cancel a pending swap:

```bash
curl -X POST "$API_BASE/swaps/$SWAP_ID/cancel" \
  -H "Authorization: Bearer $REMITFLEX_API_KEY" \
  -H "Idempotency-Key: $(uuidgen)"
```

Only swaps in `awaiting_deposit` can be cancelled.

## Fees

Each pair has a fee in basis points (`feeBps`) returned on the swap object. The effective output is `amount × rate` minus fees.

See [Swaps](/api-reference/overview) in the API reference for the full response schema.
